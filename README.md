# Bitcoin-Wallet
A Basic Bitcoin Wallet Written In Python

## Features

- Send Bitcoin
- Receive Bitcoin
- Manage Balance
- Export Wallet


## setup

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.

```bash
pip install -r requirements.txt
```
